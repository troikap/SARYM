Módulo de Backup y Recuperación

Usted podrá generar un Backup de toda la Base de Datos del sistema y recuperar la información de un un archivo de Backup previamente generado.
Se recomienda no realizar acciones con usuarios conectados.

Presione el siguiente botón para generar un archivo de Backup de todo el sistema. El mismo puede demorar unos minutos en ser generado.

Presione el siguiente botón para cargar un archivo de Backup del sistema, previamente generado. El mismo puede demorar unos minutos en ser cargado.

¡Precación!
La siguiente acción reemplazará los datos actuales del sistema. Tenga en cuenta que la carga de un archivo de Backup recuperará los datos del sistema a la fecha y hora en que se generó dicho archivo. Se recomienda discreción antes de continuar.
